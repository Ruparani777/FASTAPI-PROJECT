# 📒 FastAPI Notes API

A simple REST API built using FastAPI to manage personal notes.

## 🚀 Features

- Create a note
- List all notes
- View a note by ID
- Delete a note
- Interactive Swagger docs at `/docs`

## 🛠 Tech Stack

- FastAPI
- SQLite + SQLAlchemy ORM
- Docker (optional)
